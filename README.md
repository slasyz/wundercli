
Commands:
---------

  - Show list of lists:
    ```
    $ wundercli list
    ```

  - Show list items:
    ```
    $ wundercli show [LISTNAME]
    ```

  - Create new list:
    ```
    $ wundercli create [LISTNAME]
    ```

  - Remove the list:
    ```
    $ wundercli remove [LISTNAME]
    ```

  - Add item to a list:
    ```
    $ wundercli append [LISTNAME [ITEMTEXT]]
    ```

  - Mark item checked:
    ```
    $ wundercli check [LISTNAME]
    ```

  - Edit item:
    ```
    $ wundercli edit [LISTNAME]
    ```
